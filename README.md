# Ez Paint

A Python paint app that lets you draw with colors and erase on multiple layers.

## Description

A simple paint app, made with python using customtkinter, to draw and erase in multiple layers and save your artwork.

## How to Use

|  Toolbar Elements  | Description                                  |
| :-----------------: | -------------------------------------------- |
| Brush Color Display | Select Color of Brush                        |
|  Brush Size Slider  | Drag Slider to Adjust Size of Brush          |
|    Layer Spinbox    | Select a Layer                               |
| Clear Layer Button | Delete all Brush Strokes from Selected Layer |
|  Clear All Button  | Delete all Brush Strokes from All Layers     |
|     Save Button     | Save Artwork as .png                         |

| Color Picker Elements | Description                        |
| :-------------------: | ---------------------------------- |
|      Color Wheel      | Select Hue and Saturation of Color |
|     Value Slider     | Select Brightness of Color         |

| Canvas Controls | Action                |
| :-------------: | --------------------- |
|   Left-Click   | Paint                 |
|   Right-Click   | Erase                 |
|  Ctrl + Scroll  | Adjust Brush Size     |
| Shift + Scroll | Scroll through Layers |
